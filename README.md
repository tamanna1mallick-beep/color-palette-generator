# 🎨 Color Palette Generator

A clean, fast, and free online tool to generate beautiful color palettes for designers, developers, and creatives.

**Built for [Digital Heroes](https://digitalheroeso.com)**

---

## ✨ Features

- 🎯 Generate harmonious color palettes instantly
- 📋 Copy HEX codes with one click
- 🎨 Export palettes in JSON, CSS, and Tailwind formats
- ⚡ Fast, lightweight, and works on all devices
- 💯 100% free — no signup required

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

---

## 🚀 Live Demo

## 🙋‍♂️ Why I Built This

I built this while learning frontend development to practice JavaScript and CSS. I often need color palettes for my web projects and found existing tools either too complex or behind paywalls. I wanted something simple, fast, and completely free that works instantly without any signup.
